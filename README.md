# Twitch Stream Assistant with Web Dashboard.

Building a twitch stream assistant with a web dashboard that listens and responds to chat with the support of custom commands set by the streamer (eg `!hello`, `!save`, `!bravo`) 

## Tech stack
* Bot: `Python , twitchio`
* Backend API : `Python, FastAPI`
* Frontend: `React`

## Setup
`cp example.env .env`
